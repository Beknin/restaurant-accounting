**База данных для автоматизации HR-процессов** - реляционная система, разработанная для упрощения управления персоналом, учёта рабочего времени, расчёта рейтингов и начисления заработной платы.

**Управление сотрудниками** - приём на работу, увольнение, хранение персональных данных, должностей и истории трудоустройства.
**Учёт посещаемости** - фиксация рабочих часов, опозданий, пропусков и переработок.
**Расчёт рейтинга сотрудников** - автоматическая оценка эффективности на основе данных о пропусках и переработках.
**Управление выходными и отпусками** - отслеживание запланированных и фактических дней отдыха, контроль лимитов и пересечений.
**Расчёт заработной платы** - формирование выплат на основе отработанных часов, тарифных ставок и бонусных начислений.
